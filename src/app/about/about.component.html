Project Summary

In this project, an user can add the list of employees and can manage them with ease. This project is built on Angular framework which is responsible for frontend. The backend part of this project has used Java Spring Boot. The built tool is Maven Project. For database it has used MySQL Database.

In this project all four CRUD operations are performed i.e. CREATE, UPDATE, READ and DELETE. I have used JPA dependency which is resposible for CRUD operations. It consist of ready made funtions to perform actions on database.

In database it has used a single table user_pragati to store all data of the employees. It consist of following attributes:
  • usersignupid
  • firstname
  • lastname
  • address
  • pincode

In the table, usersignupid is a primary key which is a random number generated by the random funtion. The usersignupid is unique in the table. The DELETE and UPDATE operations are formed by fetching data from table by usersignupid
